# Credit Card Customer Segmentation

## Project Overview

This project segments credit card customers into meaningful groups using unsupervised machine learning.

## Dataset

- Source: Kaggle
- Rows: 8950
- Features: 18

## Objectives

- Perform EDA
- Engineer new features
- Build preprocessing pipeline
- Apply multiple clustering algorithms
- Compare clustering performance
- Interpret customer segments

## Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn

## Algorithms

- KMeans
- Hierarchical Clustering
- Gaussian Mixture Model
- DBSCAN

## Evaluation

- Silhouette Score
- Davies-Bouldin Index

## Best Model

KMeans
